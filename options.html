NaviGestures — Settings

NaviGestures is a free and open-source extension focused on privacy and minimal resource usage.

There are no ads, no tracking, and no telemetry or data collection of any kind. Ever.

Source code: https://github.com/NaviCorpOSS/NaviGestures

Documentation: https://github.com/NaviCorpOSS/NaviGestures/wiki

Homepage: https://oss.navicorp.net/NaviGestures

Each action uses a taught shape or the default direction path. Click a gesture preview to record, replace, or restore the default; the diagram shows direction from the start dot. Press Save so tabs pick up changes.
